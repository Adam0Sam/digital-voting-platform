Directions to access product or any other additional information

Install the npm dependencies of the root project and then install the npm dependencies of both the server and client modules in the apps workspace and the npm dependencies of the ambassador module in the packages workspace.

Next, setup the environment variables in the .env file of the server module. JWT_PUBLIC_KEY should be the public key which decodes and verifies encrypted JWT tokens used for authentication.

Next, setup a database. This may either be a local database using docker or a cloud database (like supabase). In both cases, a DATABASE_URL must be provided for a connection between the prisma client and the database.

Next, in the server module, run `npx prisma migrate dev` to upload the prisma schema to the database and in the ambassador module, run `npm build`.

Finally, in the root of the application, run `npm run dev` to start both the server and client modules.
